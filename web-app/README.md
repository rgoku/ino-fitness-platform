# Web App Installation & Running

After installing Node.js, run these commands from this directory (`web-app`):

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open in browser
# Navigate to http://localhost:3000
```

## Building for Production

```bash
npm run build
npm run start
```

## Features
- All 6 screens: Home, Reminders, Diet, Progress, Coach, Profile
- Fully responsive design
- Tailwind CSS styling
- React state management
- Interactive components
