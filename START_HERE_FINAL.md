## 🚀 3-Step Launch Path

### Step 1: Local Setup (30 min)
```bash
# Clone frontend
npx degit 0xFrann/2025-fitness-pro-complete-final FitnessPro

# Clone backend
npx degit 0xFrann/fitness-pro-backend-2025-final backend

# Setup both (see PRODUCTION_SETUP_GUIDE.md for details)
npm install        # frontend
pip install -r requirements.txt  # backend

# Get API keys:
# - Supabase: https://app.supabase.com
# - Claude: https://console.anthropic.com
# - Gemini: https://ai.google.dev

# Test locally
npx expo start     # Terminal 1 - frontend
uvicorn main:app --reload  # Terminal 2 - backend
```

### Step 2: Deploy Backend (15 min)
```bash
# Deploy to Railway (1 click)
cd backend
railway up

# Get URL: https://fitness-backend-xyz.railway.app
# Update frontend .env with this URL
```

### Step 3: Build & Ship (30 min)
```bash
# Build for iOS
eas build --platform ios --release
# Submit to App Store

# Build for Android
eas build --platform android --release
# Submit to Google Play
```

**You're live in less than 1.5 hours. 🎉**
