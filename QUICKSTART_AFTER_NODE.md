# 🚀 QUICK START (After Node.js Installation)

## The Only 3 Commands You Need

### Terminal 1: Mobile App
```powershell
cd "c:\Users\MINI\Desktop\INO_FITNESS_APP\trainer-app\apps\mobile"
npx expo install expo-router @expo/vector-icons react-native-safe-area-context react-native-screens
npm install
npx expo start
```
Then press `i` for iOS or `a` for Android, or scan QR with Expo Go app.

### Terminal 2: Web App
```powershell
cd "c:\Users\MINI\Desktop\INO_FITNESS_APP\web-app"
npm install
npm run dev
```
Then open http://localhost:3000 in your browser.

---

## What You'll See

**Mobile:** 6 tabs with modern design (Home, Reminders, Diet, Progress, Coach, Profile)
**Web:** Same 6 screens in browser with Tailwind styling

---

## Before Running These Commands

**Install Node.js first:**
1. Go to https://nodejs.org
2. Download LTS v20.x
3. Run installer
4. Restart computer
5. Restart PowerShell

Verify: `node --version` should show v20.x.x
